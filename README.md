# Tic-Tac-Toe

A simple web-based Tic-Tac-Toe game built with HTML, CSS, and JavaScript.

## Features

- Play against the computer (AI).
- Responsive 3x3 grid game board.
- Visual turn indicator.
- "New game" button to reset the board.
- Color-coded player moves (X in red, O in blue).

## Getting Started

1. Clone or download the repository.
2. Open `index.html` in your web browser.

## File Structure

- `index.html` – Main HTML file containing the game layout.
- `tictactoe.css` – Styles for the game board and elements.
- `tictactoe.js` – Game logic and interactivity.
- `README.md` – Project information and instructions.

## How to Play

- Click on any empty cell to make your move as "X".
- The computer will automatically make its move as "O".
- The game announces the winner or a draw.
- Click "New game" to start over.
